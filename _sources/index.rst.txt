.. module:: alphagradient

***************************
AlphaGradient Documentation
***************************

**Date**: |today| | **Version**: 0.0.2

**Useful links**:
`Source Repository <https://github.com/nathanheidacker/alphagradient>`__ 

:mod:`AlphaGradient` is a python package for quickly building and iterating on
financial algorithms. AlphaGradient is open source and uses the Apache 2.0
software license.

.. note::
	This is a very early, pre-release build of AlphaGradient. Unit tests are for
	all modules are still under active development; stability issues are expected.

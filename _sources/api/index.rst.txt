=============
API Reference
=============

This document outlines all of the classes, functions, and methods exposed through the top-level ``alphagradient`` module, as well as public submodules including ``alphagradient.dtypes`` and ``alphagradient.utils```.

.. warning::

    Currently, top-level modules ``alphagradient.data`` and ``alphagradien.models`` are **private**, and do not guarantee stable performance.
